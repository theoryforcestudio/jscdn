Below you can find few useful search engines. I have turned their interfaces into english to make them comprehensible for international visitors. Please learn some polish words that help you when travelling further:
- szukaj - search,
- zaawansowane - advanced,
- pomoc - help,
- plik - file,
- katalog - directory,
- haslo - password,
- wlacz - to turn on,
- wylacz - to turn off.

Btw, please note that google is also very useful for .pl searches...
Well, good hunting!
netsprint
Don't underestimate the power of Netsprint! This young Google-type search engine has indexed almost 7 million polish pages (as at end of January 2002). If you are going to use nuances of polish language it is probably to best one.
Use or:, -, site:, url:, link:, keywords:, description:, title: in its clear interface.

onet
Very good (powered by Inktomi) and fast search engine. Because its simple search is as useful as Altavista's one, if you don't know polish you may want to use an interface below. Please remember that (depending what you are looking for...) you may wish to turn a parential control off (it is off if you see "Filtr Rodzinny jest wylaczony", othrwise use "wylacz" to get rid of it.
