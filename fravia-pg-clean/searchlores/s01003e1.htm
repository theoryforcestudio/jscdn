An example:
Merlin
The 2.700.000 bytes long "Merlin" cab file is the package containing Microsoft Agent. You may be excused if you never heard of this software set.
Microsoft Agent displays animated characters in their own windows, a window that always appear at the top of the window z-order (that is, always on top). A user can move a character's window by dragging the character with the left mouse button. This is supposeed to be (in americans' minds) educational and fun. The character image moves with the pointer. In addition, an application can move a character per code using the MoveTo method.

In addition to spoken audio output, the Microsoft Agent interface supports "textual captioning" in the form of text output in cartoon-style word balloons. Words appear in the balloon as they are spoken. The balloon hides once spoken output has been completed.

In other words: Microsoft Agent is a set of softwares supporting Text-To-Speech, Voice Recognition and "text balloons". With the Voice Synthesis, the "character" can speak in different languages and produce at the same time the text you write in sound files. He can play directly WAV format files as well. With Voice Recognition, your character understands you and reacts to your voice orders. People are supposed to use this "as a guide for site presentation" or as "a virtual companion", whatver this is supposed to mean. The balloon reproduces the text the character says.

Note that in order to use Microsoft agent you will need as well
  • AgentControl (MSAgent.exe 395 Ko), which is the ActiveX control that will give you access to the properties of the Agent.
  • TrueVoice - (tv_enua.exe 997 Ko): Lernout & Hauspie control that converts texts writing in English to sound format.
Once these components installed, supposed you already have a "character", you still have to visit a web page or an application using Microsoft Agent technology (see below for a small list).

It is unfortunately not yet finished: to "enjoy" Microsoft Agent you will also need
  • Microsoft Command and Control Engine (actcnc.exe 8363 Ko) if you want to use the voice recognition (only in English, nota bene)
  • Speech Control Panel (spchcpl.exe 840Ko), with this you can list the vocal synthesis present into your system
  • Linguistic Inormation Sound Editing Tool (liset.exe 6,3 Mo), you will need it in order to create "custom speech files"

Actually you have always had only a (big) PART of it (Godzilla knows why), because as I have pointed above, you need additional components to truly "exploit" Microsoft Agent, should you decide to do so against all sound advice.

How to Extract Compressed Windows Files (cab files)
The Microsoft CAB (short for Cabinet) format is used by most Microsoft setup programs, including those used to install Windows 95, 98 and NT. CAB files provide both file compression and file grouping.
Basically, *.cab files are just compressed files, like zip files. You have to use the extract.exe program (in your windoze's directory) to peruse and extract files from them.

Internet Explorer 4.0 and later, and Windows 98, use a new compression algorithm for compressed cabinet files. You cannot extract files from Internet Explorer 4.0 and later or Windows 98 cabinet files using the version of Extract.exe included with Windows 95. You must use the version of Extract.exe included with Internet Explorer 4.0 or later or Windows 98.

You can use the Extract tool to list the contents of cabinet files without actually extracting any files. To display the contents of a cabinet file, use the following syntax: 
extract /d <cabinet>
 If you do not know which cabinet file contains a specific file you want to extract, you may use the following command to search all the cabinet files in sequential order and then extract the file once it is found: 
extract /a <cabinet> <filename> /l <destination>
