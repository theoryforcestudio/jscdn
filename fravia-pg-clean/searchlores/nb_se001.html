How quotas limit server load
=============================
Casual users on this server are limited to a certain maximum number of bytes, and a certain maximum number of messages. Access to pages at www.healthnet.org are not counted towards quota.
Quotas are currently 2 messages for most users during the course of a week. This may be altered at any time due to changes in system load.
Quotas are regenerated after seven full days; for example, if you run out of quota on a Wedensday, you will be able to use the system again the next Wedensday. If you run out on a Friday, you will have to wait until the following Friday.

6. Using DownloadSlave
visit DownLoadSlave
With Downloadslave you can conveniently download various files from the internet. The program supports HTTP and FTP protocols. When you send a request to downloadslave.com, their server will download the required files und send them to you via e-mail. Furthermore all requested files are checked for viruses. Should a virus be found, we will try to remove it from the files. In any case you will receive a warning via e-mail.
Access also possible via a freeware-software called URLRequester

Short instruction for advanced users
Ordering a password for Downloadslave Send an e-mail with the subject REMEMBER PASSWORD to data@downloadslave.com
Requesting files via e-mail Send an e-mail (without subject) to data@downloadslave.com. Put in the complete URLs (including http:// or ftp://) of the requested files. The mail has to contain your password. 
Requesting  files via URLRequester Enter your personal data (e-mail address, password, proxy etc.) into the properties field. Surf the web with Internet Explorer or Netscape Navigator as usual. Just pull every link you want to download via drag & drop into the URLRequester window instead of clicking it. Alternatively you can also type the URLs in the URLRequester window.



7. Using GrabPage
visit GrabPage
GrabPage.com is a free service that allows you to get a web page e-mailed directly to your personal e-mail device. All you do is send mail to page@grabpage.com with a Subject: line that consists of the keyword "URL:" followed by the address of the web page you want. GrabPage.com will retrieve that web page for you, format it in a way that your personal e-mail device can display, and mail it back you.

To try it online, just click on the GrabPage.com logo at the top of their page.

GrabPage is designed to return a single page. If you are interested in extracting entire web sites for later browsing, they suggest you check out WebZIP . which needs MS-IE, BTW :-((

Most personal e-mail devices aren't capable of displaying graphics, so most pictures, images and icons end up getting filtered out. Likewise, the number of columns most personal e-mail devices can display are less than what you get on your desktop computer screen, so unless you use their premium services, they'll send you back 80 columns of text per line. However, the essential information that you need out of most web pages is in the text, not the banners and fancy graphics.

See the link to the Premium Services to find out more about GrabPage's "extra features".

8. Not quite the same, but useful too : Using NetMind
There at NetMind you can sign in (for free) for a tracking service : 
NetMind's Mind-it is a FREE personalized tracking service used by people to track 
both business and personal information on the Web. It works using the browser and 
email that people already have -- so there is nothing to download or install to take 
advantage of this service.

Mind-it puts the user in control. S/he can decide what portion of a page to track, 
how often to get updated, and where to get notified about the change. 

Mind-it can track information on any Web page at any level of detail you want. It can 
track or "mind" the whole page, specific paragraphs, keywords, images, and links. 
Here is an explanation of each type of tracking option: 

1.Page Minding notifies you about any relevant change to the content that you can see 
with your browser (except images.) It ignores changes in HTML along with many of the 
common rotating ad banners.

2.Text Minding allows you to copy and paste any section from a page and only be notified 
when that section changes. After clicking Mind-it, you can use your mouse to select the 
text on the page and then use the copy/paste function in your browser to grab the text 
and paste it into the box. Click continue and the wizard will ask you to confirm and then 
you will be finished!
                        
3.Precision Tracking is a very specific way to persistently monitor the changes to any 
text that appears in a particular region of a Webpage. It will ignore all changes to 
content on a page, except for that which you have specifically selected.
                        
4.Image Minding allows you to monitor changes in any of the image references that are 
part of a page.
For example, you can track photos, graphs, cartoons, any image that is referenced by a 
different HTML tag. These are part of the HTML that is embedded in the page but not 
displayed in your browser. Unless you do Image Minding (or register the URL for the 
image, which requires some knowledge of HTML), Mind-it  ignores all changes to the images.
                        
5.Link Minding allows you to monitor changes in any of the hyperlinks that are part of 
a page, so that you know if it is dead or alive. These are part of the HTML that is 
embedded in the page. This selection is also useful for monitoring external links to 
your site. Unless you select this option, Mind-it ignores all changes to the HTML on a 
page, including changes to the hyperlinks themselves.
                        
6.Keyword Minding lets you use Enterprise Minder to watch for any keywords or phrases 
that you choose. Mind-it sends you an email when one of your keywords arrives, changes, 
or disappears. After selecting this option in the menu, simply follow the instructions 
in each step. To watch for a phrase (rather than just a single word), be sure to surround 
it with quotation marks. Otherwise, Enterprise Minder will think that each word that you 
type is supposed to be a separate keyword. (Note: This is also important if you are watching 
for simple words that may show up in a lot of ways. 
For example, watching for just the word "sun" will result in notification for the appearance 
of sunny and Sun Microsystems, while watching for "Sun Microsystems" or just "Sun ", with 
the quotes and a space at the end of the word, will eliminate notification for the word, "sunny."
